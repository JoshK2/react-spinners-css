import { Ellipsis } from './ellipsis';

Default color:

```js live
<Ellipsis />
```

With custom color:

```js live
<>
  <Ellipsis color="#be97e8" />
  <Ellipsis color="#54f1d2" />
</>
```

Different sizes:

```js live
<>
  <Ellipsis />
  <Ellipsis size={32} />
  <Ellipsis size={16} />
</>
```
