import { Ripple } from './ripple';

Default color:

```js live
<Ripple />
```

With custom color:

```js live
<>
  <Ripple color="#be97e8" />
  <Ripple color="#54f1d2" />
</>
```

Different sizes:

```js live
<>
  <Ripple />
  <Ripple size={32} />
  <Ripple size={16} />
</>
```
